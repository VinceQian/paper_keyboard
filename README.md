# Paper Keyboard

This repository contains prototypes and course materials for a paper-based semi-virtual keyboard system.

The project explores how to use computer vision and sensor signals to detect finger positions, recognize possible key presses, and record intermediate data for teaching and experimentation.

## Current Version

- `prototype_recorder_v1/`

This version focuses on:

1. Refactoring the original prototype into modules
2. Running a basic demo
3. Recording demo data into JSON files
4. Preparing reusable data for classroom lessons
